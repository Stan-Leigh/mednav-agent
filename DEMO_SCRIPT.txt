[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
This is our agent workflow architecture. Every query from the user first enters the Security Checkpoint, which automatically scrubs personally identifiable information and verifies that the request does not ask for diagnoses or prescriptions. If it passes, it moves to the MedNav Orchestrator. The orchestrator is a coordinator agent that delegates tasks to our two specialized sub-agents: the MedJargonAgent, which simplifies medical terminology, and the ClinicalTrialAgent, which queries clinical trial registries. Both sub-agents leverage a local Model Context Protocol, or MCP, server to pull terminology definitions and clinical trial listings. Once compiled, the response passes through a Human Review gate before being delivered as the Final Response to the patient.

[3:00 — SECURITY & MCP]
(show terminal or logs)
Our Security Checkpoint outputs structured JSON audit logs for every request. By keeping the MCP tools separated in a dedicated stdio server, we ensure that our data lookup functions run safely and independently of the LLM orchestration.
